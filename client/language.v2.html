Top 20 Programming Languages used in Malaysia

Shows the frequency of the languages used based on data from Github Users in Malaysia (June 2017).

The following criteria has to be met:

  • has `Malaysia` in the location
  • only public repos are taken into account (no forked repos)
  • Early github accounts does not have the field language exposed in the api, so that is not taken into account
  • One repo can have multiple languages (e.g. javascript, html, css)
  • Private repositories are not taken into account
